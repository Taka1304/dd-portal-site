# Overview

Official website of Data Dreamers

## Getting Started

First, run the development server:

### Required

- nvm v1.1.12
- node v20.16.0
- corepack v0.29.3

```bash
pnpm install --frozen-lockfile

pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
